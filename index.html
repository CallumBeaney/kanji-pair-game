二漢字複合語狩猟
Each grid is made of 16 kanji from 8 words. Tap 2 kanji and if they turn green, you've found one. Language is tricky. The 16 kanji might have more than 8 combinations... if a green kanji turns yellow, try using it with remaining grey ones. This game is designed to be challening and is intended for intermediate-advanced speakers of Japanese.
